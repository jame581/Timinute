# MCP Server Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Host a Model Context Protocol server inside Timinute so AI assistants, authenticated with a user-generated Personal Access Token, can query and mutate that user's projects, time entries, and analytics — with every tool call recorded to a user-viewable audit log.

**Architecture:** A new `PersonalAccessToken` entity + a custom `"Pat"` authentication scheme slot into the existing `ApplicationDefinedPolicy` forward selector (bearer tokens prefixed `tmn_pat_` route to PAT, others to JWT). Shared `*AppService` classes hold the ownership/validation logic that both controllers and MCP tools call. `ModelContextProtocol.AspNetCore` exposes tools at `/mcp`; a single invocation interceptor gates write tools by token scope and writes an `McpActivityLog` row per call. A background service purges activity older than 90 days.

**Tech Stack:** .NET 10, EF Core 10 / SQL Server, ASP.NET Core auth handlers, `ModelContextProtocol.AspNetCore` (prerelease), Blazor WebAssembly (Aurora), xUnit + Moq + `WebApplicationFactory<Program>`.

**Spec:** `docs/superpowers/specs/2026-07-17-v2.4-mcp-server-design.md`

**Depends on:** the Support Logging plan (`2026-07-17-v2.4-support-logging.md`) is merged first — the activity interceptor emits correlated Serilog events.

## Global Constraints

- Target `net10.0`. Pin the **exact** resolved version of `ModelContextProtocol.AspNetCore` in the `.csproj`; keep all MCP code under `Server/Mcp/` so the prerelease dep is isolated.
- Token format: `tmn_pat_` + Base64Url(32 CSPRNG bytes). Shown **once**; never retrievable. Only a **SHA-256 hash** + an 8-char `Prefix` are stored — never the plaintext, never a password hasher.
- The user-id claim controllers read is `Constants.Claims.UserId` (= `"sub"`). PAT principals must carry it.
- Scopes: `read` and `read_write`. Write tools require `read_write`; enforce centrally, not per-tool.
- Every MCP tool call (read and write) writes exactly one `McpActivityLog` row (Time, Tool, Result, Summary, optional Detail, TokenId). Retained 90 days.
- `/mcp` gated by `Mcp__Enabled` (default `true`) and requires the `"Pat"` scheme; inert without a valid PAT.
- Never modify existing JWT/cookie auth behavior. `PatController` and the settings pages are reachable only under the existing cookie/JWT schemes — **a PAT can never mint or list tokens**.
- All new dates are `DateTimeOffset` UTC (house style). Reuse `IRepositoryFactory`/`IRepository<T>` and `MappingProfile`.

`PatTokenService.cs`:

```csharp
using System.Security.Cryptography;
using System.Text;

namespace Timinute.Server.Services.Pat
{
    // Tokens are 256 bits of CSPRNG output, so a single SHA-256 (fast) is the correct
    // at-rest transform — a password hasher (bcrypt/PBKDF2) would only add cost with no
    // security benefit for high-entropy secrets.
    public sealed class PatTokenService : IPatTokenService
    {
        public (string plaintext, string hash, string prefix) Generate()
        {
            var bytes = RandomNumberGenerator.GetBytes(32);
            var body = Base64UrlEncode(bytes);
            var plaintext = IPatTokenService.TokenPrefix + body;
            return (plaintext, Hash(plaintext), plaintext.Substring(0, 8));
        }

        public string Hash(string plaintext)
        {
            var digest = SHA256.HashData(Encoding.UTF8.GetBytes(plaintext));
            return Convert.ToHexStringLower(digest);
        }

        public bool FixedTimeEquals(string hashA, string hashB)
            => CryptographicOperations.FixedTimeEquals(
                Encoding.UTF8.GetBytes(hashA), Encoding.UTF8.GetBytes(hashB));

        private static string Base64UrlEncode(byte[] bytes)
            => Convert.ToBase64String(bytes).TrimEnd('=').Replace('+', '-').Replace('/', '_');
    }
}
```

Note: `plaintext.Substring(0, 8)` = `"tmn_pat_"` for every token, so `Prefix` alone is not a lookup discriminator. **Correction:** use the first 8 chars *of the random body* for the prefix. Update `Generate()`:

```csharp
var prefix = body.Substring(0, 8);
return (plaintext, Hash(plaintext), prefix);
```

and update the test's prefix assertions to `Assert.Equal(8, prefix.Length);` and `Assert.False(prefix.StartsWith("tmn_pat_"));` (drop the `plaintext.Substring(0,8)` equality). Re-run.

```csharp
using System.Security.Claims;
using System.Text.Encodings.Web;
using Microsoft.AspNetCore.Authentication;
using Microsoft.EntityFrameworkCore;
using Microsoft.Extensions.Options;
using Timinute.Server.Data;
using Timinute.Server.Helpers;
using Timinute.Server.Models;
using Timinute.Server.Services.Pat;

namespace Timinute.Server.Auth
{
    public sealed class PatAuthenticationHandler : AuthenticationHandler<AuthenticationSchemeOptions>
    {
        public const string SchemeName = "Pat";

        private readonly ApplicationDbContext db;
        private readonly IPatTokenService tokens;

        public PatAuthenticationHandler(
            IOptionsMonitor<AuthenticationSchemeOptions> options, ILoggerFactory logger,
            UrlEncoder encoder, ApplicationDbContext db, IPatTokenService tokens)
            : base(options, logger, encoder)
        {
            this.db = db;
            this.tokens = tokens;
        }

        protected override async Task<AuthenticateResult> HandleAuthenticateAsync()
        {
            string? authorization = Request.Headers.Authorization;
            if (string.IsNullOrWhiteSpace(authorization) ||
                !authorization.StartsWith("Bearer ", StringComparison.OrdinalIgnoreCase))
                return AuthenticateResult.NoResult();

            var raw = authorization.Substring("Bearer ".Length).Trim();
            if (!raw.StartsWith(IPatTokenService.TokenPrefix, StringComparison.Ordinal))
                return AuthenticateResult.NoResult();

            var body = raw.Substring(IPatTokenService.TokenPrefix.Length);
            if (body.Length < 8)
                return AuthenticateResult.Fail("Malformed token.");

            var prefix = body.Substring(0, 8);
            var hash = tokens.Hash(raw);

            var candidates = await db.PersonalAccessTokens
                .Where(t => t.Prefix == prefix).ToListAsync();

            var now = DateTimeOffset.UtcNow;
            var match = candidates.FirstOrDefault(t =>
                tokens.FixedTimeEquals(t.TokenHash, hash)
                && t.RevokedAt == null
                && (t.ExpiresAt == null || t.ExpiresAt > now));

            if (match is null)
                return AuthenticateResult.Fail("Invalid token.");

            // Best-effort last-used stamp; must never fail the request.
            try
            {
                match.LastUsedAt = now;
                await db.SaveChangesAsync();
            }
            catch { /* non-fatal */ }

            var scope = match.Scopes == PatScope.ReadWrite ? "read_write" : "read";
            var identity = new ClaimsIdentity(new[]
            {
                new Claim(Constants.Claims.UserId, match.UserId),
                new Claim(Constants.Claims.Scope, scope),
                new Claim("pat_id", match.Id),
            }, SchemeName);

            return AuthenticateResult.Success(
                new AuthenticationTicket(new ClaimsPrincipal(identity), SchemeName));
        }
    }
}
```

```csharp
using Microsoft.AspNetCore.Authorization;
using Microsoft.AspNetCore.Mvc;
using System.Security.Claims;
using Timinute.Server.Helpers;
using Timinute.Server.Models;
using Timinute.Server.Repository;
using Timinute.Server.Services.Pat;
using Timinute.Shared.Dtos.Pat;

namespace Timinute.Server.Controllers
{
    [Authorize]                 // cookie/JWT via the policy scheme — NEVER the Pat scheme
    [ApiController]
    [Route("[controller]")]
    public class PatController : ControllerBase
    {
        private readonly IRepository<PersonalAccessToken> repo;
        private readonly IPatTokenService tokens;
        private readonly ILogger<PatController> logger;

        public PatController(IRepositoryFactory factory, IPatTokenService tokens, ILogger<PatController> logger)
        {
            repo = factory.GetRepository<PersonalAccessToken>();
            this.tokens = tokens;
            this.logger = logger;
        }

        [HttpGet]
        public async Task<ActionResult<IEnumerable<PersonalAccessTokenDto>>> List()
        {
            var userId = User.FindFirstValue(Constants.Claims.UserId);
            if (string.IsNullOrEmpty(userId)) return Unauthorized();

            var all = await repo.GetAll(t => t.UserId == userId && t.RevokedAt == null);
            return Ok(all.Select(ToDto));
        }

        [HttpPost]
        public async Task<ActionResult<CreatedPatDto>> Create([FromBody] CreatePatDto dto)
        {
            var userId = User.FindFirstValue(Constants.Claims.UserId);
            if (string.IsNullOrEmpty(userId)) return Unauthorized();

            var (plaintext, hash, prefix) = tokens.Generate();
            var entity = new PersonalAccessToken
            {
                UserId = userId,
                Name = dto.Name.Trim(),
                TokenHash = hash,
                Prefix = prefix,
                Scopes = dto.Scope == "read_write" ? PatScope.ReadWrite : PatScope.Read,
                CreatedAt = DateTimeOffset.UtcNow,
                ExpiresAt = dto.ExpiresAt
            };
            await repo.Insert(entity);
            logger.LogInformation("PAT {Prefix} ({Scope}) created for user {UserId}.", prefix, entity.Scopes, userId);

            return Ok(new CreatedPatDto { Id = entity.Id, Token = plaintext, Prefix = prefix, Scope = dto.Scope });
        }

        [HttpDelete("{id}")]
        public async Task<ActionResult> Revoke(string id)
        {
            var userId = User.FindFirstValue(Constants.Claims.UserId);
            if (string.IsNullOrEmpty(userId)) return Unauthorized();

            var entity = await repo.GetById(id);
            if (entity == null || entity.UserId != userId || entity.RevokedAt != null) return NotFound();

            entity.RevokedAt = DateTimeOffset.UtcNow;
            await repo.Update(entity);
            return NoContent();
        }

        private static PersonalAccessTokenDto ToDto(PersonalAccessToken t) => new()
        {
            Id = t.Id, Name = t.Name, Prefix = t.Prefix,
            Scope = t.Scopes == PatScope.ReadWrite ? "read_write" : "read",
            CreatedAt = t.CreatedAt, LastUsedAt = t.LastUsedAt, ExpiresAt = t.ExpiresAt
        };
    }
}
```

```csharp
using Timinute.Server.Models;

namespace Timinute.Server.Mcp
{
    // Wraps every tool call: run it, then record one activity row + a correlated Serilog event.
    public sealed class McpActivityInterceptor
    {
        private readonly McpActivitySink sink;
        private readonly McpUserContext user;
        private readonly ILogger<McpActivityInterceptor> logger;

        public McpActivityInterceptor(McpActivitySink sink, McpUserContext user, ILogger<McpActivityInterceptor> logger)
        {
            this.sink = sink; this.user = user; this.logger = logger;
        }

        public async Task<TResult> RunAsync<TResult>(string toolName, Func<Task<TResult>> next, CancellationToken ct)
        {
            try
            {
                var result = await next();
                await sink.WriteAsync(user.UserId, user.TokenId, toolName,
                    $"{toolName} ok", McpActivityResult.Success, null, ct);
                logger.LogInformation("MCP tool {Tool} succeeded for user {UserId}", toolName, user.UserId);
                return result;
            }
            catch (Exception ex)
            {
                await sink.WriteAsync(user.UserId, user.TokenId, toolName,
                    $"{toolName} failed", McpActivityResult.Failed, ex.Message, ct);
                logger.LogError(ex, "MCP tool {Tool} failed for user {UserId}", toolName, user.UserId);
                throw;
            }
        }
    }
}
```

```csharp
using Microsoft.EntityFrameworkCore;
using Timinute.Server.Data;

namespace Timinute.Server.Services
{
    public class McpActivityPurgeService : BackgroundService
    {
        private readonly IServiceScopeFactory scopeFactory;
        private readonly IConfiguration configuration;
        private readonly ILogger<McpActivityPurgeService> logger;

        public McpActivityPurgeService(IServiceScopeFactory scopeFactory, IConfiguration configuration, ILogger<McpActivityPurgeService> logger)
        {
            this.scopeFactory = scopeFactory; this.configuration = configuration; this.logger = logger;
        }

        protected override async Task ExecuteAsync(CancellationToken stoppingToken)
        {
            var intervalHours = Math.Max(1, configuration.GetValue<int>("Mcp:ActivityRetention:PurgeIntervalHours", 24));
            using var timer = new PeriodicTimer(TimeSpan.FromHours(intervalHours));
            await RunOnce(stoppingToken);
            while (!stoppingToken.IsCancellationRequested && await timer.WaitForNextTickAsync(stoppingToken))
                await RunOnce(stoppingToken);
        }

        private async Task RunOnce(CancellationToken ct)
        {
            try
            {
                var retentionDays = configuration.GetValue<int>("Mcp:ActivityRetention:Days", 90);
                using var scope = scopeFactory.CreateScope();
                var db = scope.ServiceProvider.GetRequiredService<ApplicationDbContext>();
                var count = await PurgeOnce(db, retentionDays, ct);
                logger.LogInformation("McpActivityPurge: purged {Count} activity rows older than {Days}d", count, retentionDays);
            }
            catch (Exception ex) { logger.LogError(ex, "McpActivityPurge tick failed"); }
        }

        public static async Task<int> PurgeOnce(ApplicationDbContext db, int retentionDays, CancellationToken ct)
        {
            var cutoff = DateTimeOffset.UtcNow.AddDays(-retentionDays);
            var old = await db.McpActivityLogs.Where(a => a.Timestamp < cutoff).ToListAsync(ct);
            db.McpActivityLogs.RemoveRange(old);
            await db.SaveChangesAsync(ct);
            return old.Count;
        }
    }
}
```
